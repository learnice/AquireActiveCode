## AquireActiveCode

获取一个临时 IDEA 注册码

在 **Releases** 中下载最新的 **AquireActiveCode-GUI.jar** 到本地，**双击运行**，然后将复制的激活码粘贴到 IDEA 中激活即可（JetBrains 其它软件亦同）

IDEA 提示过期后需要重复上面操作

---


Get a temporary IDEA activation code

Download the latest **AquireActiveCode-GUI.jar** from **Releases** to the local, **double-click** to run,Then paste the copied activation code into IDEA to activate it（JetBrains other software is the same）

IDEA prompts that the above operation needs to be repeated after expiration
